# ThreeDfiy-api
A RESTful API for ThreeDfiy.

# What's ThreeDfiy?
ThreeDfiy is a online platform where you can upload images and create a 3D reconstruction of the images.

## Installation
```bash
$ python -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
```

## Run
```bash
$ python src/main.py
```
